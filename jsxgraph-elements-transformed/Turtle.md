

# Turtle



## 描述

A turtle is a graphic paradigm similar to the programming languages Logo or PostScript.

### 说明

This element has no direct constructor. To create an instance of this element you have to call [JXG.Board#create](./JXG.Board.md#create) with type "turtle".  
  

Possible parent array combinations are:

{[JXG.Board](./JXG.Board.md)} **board**
:   The board the turtle is put on.
  
  

{Array} **parents**
  
  

{Object} **attributes**
:   Object containing properties for the element such as stroke-color and visibility. See [JXG.GeometryElement#setAttribute](./JXG.GeometryElement.md#setAttribute)
