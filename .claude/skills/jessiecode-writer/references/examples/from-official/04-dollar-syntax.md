# 美元符号语法 (Dollar Syntax)

## 图形描述

本示例演示了 JessieCode 中使用 `$()` 语法通过元素 ID 引用已有元素的方式。这种语法允许动态访问和修改元素的属性。

## JessieCode 代码

```js
A = point(1, 1) << id: 'p1' >>;
$('p1').fillColor = 'green';

f = function () {
    return ($('p1').X()==1&&$('p1').Y()==1?(true):(false));
};
```

## 知识点

1. **元素 ID**：创建元素时使用 `<< id: 'p1' >>` 为元素指定唯一标识符
2. **美元符号引用**：`$('p1')` 通过 ID 引用已有元素
3. **属性访问与修改**：`$('p1').fillColor = 'green'` 动态修改元素属性
4. **坐标方法**：`.X()` 和 `.Y()` 获取点的 x 和 y 坐标
5. **条件表达式**：使用三元运算符 `? :` 进行条件判断
6. **逻辑运算**：使用 `&&` 进行逻辑与运算
