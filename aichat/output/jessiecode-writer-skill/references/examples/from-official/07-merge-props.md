# 条件表达式示例 (Conditional Expression Example)

## 图形描述

本示例演示了 JessieCode 中的条件表达式（三元运算符）和逻辑运算符的使用：

- 定义多个数值变量
- 使用逻辑与运算符 `&&` 组合多个条件
- 使用三元运算符 `? :` 进行条件判断
- 在函数中返回布尔值结果

注意：原文件名为 `conditions.html`，但实际内容是条件表达式示例。

## JessieCode 代码

```js
a = (2 + 3) * 4;
b = 3 - 2 - 1;
c = 4;
f = function(x) {
    return (((a == 2 && b == 3) && c == 4) ? (true) : (false));
};
```

## 知识点

1. **算术运算**：`(2 + 3) * 4` 演示加法和乘法的优先级
2. **连续减法**：`3 - 2 - 1` 从左到右依次计算
3. **相等比较**：`==` 运算符比较两个值是否相等
4. **逻辑与**：`&&` 运算符连接多个布尔条件
5. **三元运算符**：`(condition) ? (trueValue) : (falseValue)` 条件表达式语法
6. **函数定义**：使用 `function(x) { ... }` 定义匿名函数
7. **布尔返回值**：函数返回 `true` 或 `false`
