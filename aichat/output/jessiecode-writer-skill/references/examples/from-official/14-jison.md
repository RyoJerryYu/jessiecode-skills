# 幂运算示例 (Power Operation Example)

## 图形描述

本示例演示了 JessieCode 中的幂运算语法：

- 使用 `**` 运算符进行幂运算
- 计算 2 的 4 次方

## JessieCode 代码

```js
e = 2**4;
```

## 知识点

1. **幂运算符**：`**` 是幂运算运算符（如 `2**4` 表示 2 的 4 次方）
2. **数值计算**：`2**4 = 16`
3. **JavaScript 风格**：与 JavaScript 的幂运算语法一致
