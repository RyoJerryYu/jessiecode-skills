# 赋值语法示例 (Assignment Syntax Example)

## 图形描述

本示例演示了 JessieCode 中的基本赋值语法：

- 简单变量赋值
- 连等赋值（多个变量同时赋值）
- 小数赋值

## JessieCode 代码

```js
a = 2;
b = c = d = 4.5;
```

## 知识点

1. **简单赋值**：`a = 2` 将数值 2 赋值给变量 a
2. **连等赋值**：`b = c = d = 4.5` 从右到左依次赋值，b、c、d 都等于 4.5
3. **小数**：支持十进制小数表示
4. **语句结束符**：使用分号 `;` 结束语句
