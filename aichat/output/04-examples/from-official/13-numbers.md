# 数字类型示例 (Number Types Example)

## 图形描述

本示例演示了 JessieCode 中各种数字类型和特殊值的表示：

- NaN（非数字）
- 普通整数和小数
- 正无穷大（两种表示法）
- 负无穷大（两种表示法）
- Unicode 无穷符号支持

## JessieCode 代码

```js
a = NaN;
b = 1;
c = 0.5;
d = +Infinity;
e = -Infinity;
f = ∞;
```

## 知识点

1. **NaN**：`NaN` 表示"非数字"（Not a Number）
2. **整数**：`1` 普通整数表示
3. **小数**：`0.5` 十进制小数表示
4. **正无穷大**：`+Infinity` 使用 Infinity 关键字
5. **负无穷大**：`-Infinity` 使用负号加 Infinity
6. **Unicode 无穷**：`∞` 直接使用 Unicode 无穷符号
7. **特殊数值**：支持 JavaScript 风格的特殊数值表示
