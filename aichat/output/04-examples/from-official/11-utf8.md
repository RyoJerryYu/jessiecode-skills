# UTF-8 字符支持示例 (UTF-8 Character Support Example)

## 图形描述

本示例演示了 JessieCode 对 UTF-8 字符的支持：

- 使用希腊字母作为变量值
- 支持 Unicode 字符在字符串中的使用

## JessieCode 代码

```js
alpha = 'α';
```

## 知识点

1. **UTF-8 字符串**：支持希腊字母等 Unicode 字符
2. **字符串字面量**：使用单引号包裹字符串
3. **变量命名**：变量名可以使用拉丁字母（alpha），存储 UTF-8 字符（α）
